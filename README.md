# AI Learning Assistant

## AI-Driven Emotion Detection & Personalized Learning Support Platform

This project analyzes student input, detects emotions such as Confused, Frustrated, Curious, Bored, and Confident, and provides personalized AI-based learning guidance using Python and Streamlit.

## Technologies Used
- Python
- Streamlit
- Machine Learning
- Pandas
- Scikit-learn
