# Requirement Analysis

## Functional Requirements

- Student enters study-related text.
- Detect emotion using the trained BiLSTM model.
- Generate personalized learning guidance using Google Gemini AI.
- Store user interaction history.
- Display analytics dashboard with emotion statistics.

## Non-Functional Requirements

- User-friendly interface
- Fast emotion prediction
- Accurate AI responses
- Secure API integration
- Responsive Streamlit web application

## Software Requirements

- Python 3.x
- Streamlit
- TensorFlow
- Keras
- Pandas
- NumPy
- Plotly
- Google Gemini API
- Scikit-learn

## Hardware Requirements

- Windows 10/11
- Minimum 8 GB RAM
- Internet connection
